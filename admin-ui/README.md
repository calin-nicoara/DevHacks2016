# e-carrefour-web

First time:

Run in your project root:

1. --npm install--
2. --bower install--
3. --gulp install--

For build:

1. ```gulp compile```
2. ```gulp```
--project auto reloads when it's modified--

To add new dependences:
    --for tools--
    npm install --save [dependency-name]
    
    --for libraries-- 
    bower install --save [dependency-name] (they also need to be incuded in index.html)
